# 🔥 Calories Burn Prediction App

A Machine Learning web application that predicts calories burned based on user inputs like age, gender, duration, heart rate, and body temperature.

---

## 🚀 Features

* Predict calories burned in real-time
* Built using XGBoost (high accuracy model)
* Interactive UI with Streamlit
* Clean and simple user interface

---

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* XGBoost
* Streamlit

---

## 📊 Model Details

* Algorithm: XGBoost Regressor
* Features used:

  * Gender
  * Age
  * Height
  * Weight
  * Duration
  * Heart Rate
  * Body Temperature

---

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
